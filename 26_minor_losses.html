Minor losses

  1. Pipe entrance or exit
  2. Sudden expansion or contraction
  3. Bends, elbows, tees, and other fittings
  4. Valves, open or partially closed
  5. Gradual expansions or contractions

Minor loss occurs locally across component

Recommended installation of flow meter 10-20 pipe diameters downstream

Turbulent eddies (generated by component) dissipate

Loss coefficient

  • Losses correlated with pipe flow parameters
  • Measured experimentally

Loss coefficient \[K = \dfrac{h_{l,m}}{V^2/2g} = \dfrac{\Delta P}{\frac{1}{2}\rho V^2}\]

Total pipe system loss

\[\boxed{h_{l,\text{total}} = h_l + \sum h_{l,m} = \dfrac{V^2}{2g} \left( \dfrac{f L}{D} + \sum K \right)}\]

Losses must be added separately if pipe size changes!

\[h_{l,\text{total}} = \sum f \dfrac{L}{D} \dfrac{V^2}{2g} + \sum K \dfrac{V^2}{2g}\]

Problem

  • 6-cm-diameter horizontal water pipe expands to 9-cm-diameter
  • Walls of expansion 10° from the axis
  • Average velocity and pressure before expansion is 7 m/s and 150 kPa
  • Energy correction factor is 1.06
  • Compute the head loss and pressure downstream

Solution

\[d/D = 0.06/0.09 = 6/9\] \[K = 0.133\] \[\dot{m}_1 = \dot{m}_2 \Rightarrow \rho V_1 A_1 = \rho V_2 A_2\]

\[\Rightarrow V_2 = \dfrac{A_1}{A_2} V_1^2 = \dfrac{D_1^2}{D_2^2} V_1\] \[\Rightarrow V_2 = \dfrac{0.06^2}{0.09^2} (7) = 3.11\,\mathrm{m/s}\] \[h_l = K \dfrac{V_1^2}{2g} = (0.133)\dfrac{(7)^2}{2(9.81)} = 0.333\,\mathrm{m}\]

\[\dfrac{P_1}{\rho g} + \alpha \dfrac{V_1^2}{2g} + \cancel{z_1} = \dfrac{P_2}{\rho g} + \alpha \dfrac{V_2^2}{2g} + \cancel{z_2} + h_l\] \[\Rightarrow P_2 = P_1 + \rho \left( \dfrac{\alpha V_1^2 - \alpha V_2^2}{2} - g h_l \right)\] \[=(150) + (1000)\left[ \dfrac{(1.06)(7^2-3.11^2)}{2} - (9.81)(0.333) \right]\] \[= 168\,\mathrm{kPa}\]

Problem

  • Water (ρ=1.94 slugs/ft3 and ν=0.000011 ft2/s) is pumped between two reservoirs
  • Flow rate of 0.2 ft3/s through 400 ft of 2-in-diameter pipe
  • Roughness ratio is 0.001
  • Compute the pump power required

Solution

\[\dfrac{P_1}{\rho g} + \dfrac{V_1^2}{2g} + z_1 + h_p = \dfrac{P_2}{\rho g} + \dfrac{V_2^2}{2g} + z_2 + h_l + \sum h_{l,m}\] \[P_1 = P_2 ~~~~ V_1=V_2 \approx 0\] \[h_p = z_2 - z_1 + \dfrac{V^2}{2g} \left( \dfrac{f L}{D} + \sum K \right)\]

\[V = \dfrac{Q}{A} = \dfrac{0.2}{(1/4)\pi(2/12)^2} = 9.17\,\mathrm{ft/s}\]

Loss K
Sharp entrance 0.5
Open globe valve 6.9
12-in bend 0.25
90° elbow 0.95
Half-closed gate valve 3.8
Sharp exit 1.0

\[\text{Re} = \dfrac{V D}{\nu} = \dfrac{(9.17)(2/12)}{0.000011} = 139000\] \[\epsilon/D = 0.001\] From Moody chart \(f=0.0216\) \[h_p = 120 - 20 + \dfrac{(9.17)^2}{2(32.2)} \left[ \dfrac{(0.0216)(400)}{(2/12)} + 13.4 \right]\] \[=185\,\mathrm{ft}\]

\[\dot{W}_p = \rho g Q h_p = (1.94)(32.2)(0.2)(185) = 2300\,\mathrm{ft \cdot lbf/s}\] \[\dot{W}_p = \dfrac{2300}{550} = 4.2\,\mathrm{hp}\]
