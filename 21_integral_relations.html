Sizing a pump

Review

Problem 1

  • Water to be pumped by a 78% efficient 5-kW pump
  • Diameters of pipe is 7 cm on the intake side, and 5 cm on the discharge side
  • Ignore elevation difference between pipe inlet/outlet

What assumptions can we make?

  • Flow is incompressible and steady
  • Elevation difference between pump inlet and outlet is negligible
  • Frictional effects can be ignored

What is the maximum flow rate of water?

\[\dot{W}_{\text{pump}} = \eta \dot{W}_{\text{electric}} = (0.78)(5) = 3.9\,\mathrm{kW}\] \[\dot{m} \left( \dfrac{P_1}{\rho} + \dfrac{V_1^2}{2} + gz_1 \right) + \dot{W}_{\text{pump}} = \dot{m} \left( \dfrac{P_2}{\rho} + \dfrac{V_2^2}{2} + gz_2 \right)\] \[ + \dot{W}_{\text{turbine}} + \dot{E}_{\text{loss}}\] \[\dot{W}_{\text{pump}} = \dot{m} g z_2 \Rightarrow \dot{m} = \dfrac{\dot{W}_{\text{pump}}}{g z_2}\]

\[ = \dfrac{3.9}{(9.81)(30)} = 13.25\,\mathrm{kg/s}\] \[Q = \dfrac{\dot{m}}{\rho} = \dfrac{13.25}{1000} = 0.01325\,\mathrm{m^3/s}\]

What is the pressure difference across the pump?

\[V_i = \dfrac{Q}{A_i} = \dfrac{Q}{\pi D_i^2 / 4} = \dfrac{0.01325}{\pi(0.07)^2 / 4} = 3.443\,\mathrm{m/s}\] \[V_o = \dfrac{Q}{A_o} = \dfrac{Q}{\pi D_o^2 / 4} = \dfrac{0.01325}{\pi(0.07)^2 / 4} = 6.748\,\mathrm{m/s}\]

\[\dot{m} \left( \dfrac{P_i}{\rho} + \dfrac{V_i^2}{2} + gz_i \right) + \dot{W}_{\text{pump}} = \dot{m} \left( \dfrac{P_o}{\rho} + \dfrac{V_o^2}{2} + gz_o \right)\] \[ + \dot{W}_{\text{turbine}} + \dot{E}_{\text{loss}}\] \[\Rightarrow \rho Q \left( \dfrac{P_i}{\rho} + \dfrac{V_i^2}{2} \right) + \dot{W}_{\text{pump}} = \rho Q \left( \dfrac{P_o}{\rho} + \dfrac{V_o^2}{2} \right)\]

\[ \Rightarrow P_o - P_i = \dfrac{\dot{W}_{\text{pump}}}{Q} + \dfrac{\rho (V_i^2 - V_o^2)}{2}\] \[= \dfrac{3.9}{0.01325} + \dfrac{(3.443)^2 - (6.748)^2}{2} = 294.3 - 16.8 = 277.5\,\mathrm{kPa}\]

Problem 2

  • Diameter of elbow is 10 cm
  • Weight of elbow and water in it is negligible
  • Elevation difference between the centers of the exit and the inlet of the elbow is 50 cm
  • Momentum flux correction is 1.03

What is the gage pressure at the center of the inlet of the elbow?

\[\dot{m} = 40\,\mathrm{kg/s}\] \[V_1 = V_2 = \dfrac{\dot{m}}{\rho A} = \dfrac{\dot{m}}{\rho \pi D^2 / 4} = \dfrac{40}{(1000)\pi(0.1)^2 / 4}\] \[= 5.093\,\mathrm{m/s}\]

\[\dfrac{P_1}{\rho g} + \dfrac{V_1^2}{2g} + z_1 = \dfrac{P_2}{\rho g} + \dfrac{V_2^2}{2g} + z_2\] \[P_1 - P_2 = \rho g (z_2 - z_1) \Rightarrow P_{1,\text{gage}} = \rho g (z_2 - z_1) =\] \[= (1000)(9.81)(0.50) = 4905\,\mathrm{N/m^2}\]

What is the anchoring force needed to hold the elbow in place?

\[\sum \vec{F} = \sum_{\text{out}} \beta \dot{m} \vec{V} - \sum_{\text{in}} \beta \dot{m} \vec{V}\] \[ F_{R,x} + P_{1,\text{gage}} A_1 = 0 - \beta \dot{m} V_1 = -\beta \dot{m} V \] \[ F_{R,z} = \beta \dot{m} V_2 = \beta \dot{m} V \]

\[ F_{R,x} = -\beta \dot{m} V - P_{1.\text{gage}} A_1 \] \[ = -(1.03)(40)(5.093) - (4905)(\pi)(0.1^2)/4 = -248.4\,\mathrm{N}\] \[F_{R,z} = \beta \dot{m} V = (1.03)(40)(5.093) = 209.8\,\mathrm{N}\]

\[F_R = \sqrt{F_{R,x}^2 + F_{R,z}^2} = \sqrt{(-248.4)^2 + (209.8)^2} = 325\,\mathrm{N}\] \[\theta = \tan^{-1}{\dfrac{F_{R,z}}{F_{R,x}}} = \tan^{-1}{\dfrac{209.8}{-284.4}} = -40.2\,\mathrm{^{\circ}}\]

Problem 3

  • Tank with discharge pipes accelerates to the left
  • Volumetric flow rates at both pipes are equal
  • Bernoulli equation becomes \[P_1/\rho + V_1^2/2 + gz_1 - P_2/\rho - V_2^2/2 - gz_2 = a L + \text{Losses}\]
  • What is the diameter \(D\) of the inclined pipe?
  • Ignore frictional losses

Solution

\[\left( \cancel{\dfrac{P_A}{\rho g}} + z_A + \cancelto{0}{\dfrac{V_A^2}{2g}} \right) - \left( \cancel{\dfrac{P_B}{\rho g}} + z_B + \dfrac{V_B^2}{2g} \right) = \dfrac{a}{g} \Delta L\] \[\Rightarrow h = \dfrac{V_B^2}{g} + \dfrac{a}{g} \Delta L \Rightarrow \dfrac{V_B^2}{2g} = h - \dfrac{a}{g} \Delta L\] \[\left( \cancel{\dfrac{P_A}{\rho g}} + z_A + \cancelto{0}{\dfrac{V_A^2}{2g}} \right) - \left( \cancel{\dfrac{P_C}{\rho g}} + z_C + \dfrac{V_C^2}{2g} \right) = \cancelto{0}{\dfrac{a}{g} \Delta L}\] \[\Rightarrow h = \dfrac{V_C^2}{2g}\]

\[\dfrac{V_B^2 / 2g}{V_C^2 / 2g} = \dfrac{h - \frac{a}{g} \Delta L}{h}\] \[\Rightarrow \left( \dfrac{V_B}{V_C} \right)^2 = 1 - \dfrac{a \Delta L}{gh} \Rightarrow \dfrac{V_B}{V_C} = \sqrt{1 - \dfrac{a \Delta L}{gh}}\] \[Q_B = Q_C \Rightarrow A_B V_B = A_C V_C \Rightarrow \dfrac{A_C}{A_B} = \dfrac{V_B}{V_C} = \dfrac{d^2}{D^2}\]

\[\dfrac{d^2}{D^2} = \sqrt{1 - \dfrac{a \Delta L}{g h}} = \sqrt{1 - \dfrac{(3)(8)}{(9.81)(3)}} = 0.4295\] \[\Rightarrow D = \sqrt{\dfrac{0.01^2}{0.4295}} = 0.01526\,\mathrm{m} \approx 1.53\,\mathrm{cm}\]
